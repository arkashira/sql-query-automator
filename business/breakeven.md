# breakeven.md

## Cost per Active User
- **Compute Costs**: $0.05 per hour per user
  - Average usage: 10 hours/month
  - Monthly compute cost per user: $0.05 * 10 = **$0.50**
  
- **Storage Costs**: $0.01 per GB per month
  - Average storage per user: 5 GB
  - Monthly storage cost per user: $0.01 * 5 = **$0.05**
  
- **Bandwidth Costs**: $0.02 per GB
  - Average bandwidth per user: 10 GB
  - Monthly bandwidth cost per user: $0.02 * 10 = **$0.20**

### Total Cost per Active User
- **Total Cost**: $0.50 (compute) + $0.05 (storage) + $0.20 (bandwidth) = **$0.75**

---

## Pricing Tiers
1. **Basic Tier**: $10/month
   - Features: 
     - AI-generated query suggestions
     - Basic query management
     - Email support

2. **Pro Tier**: $30/month
   - Features:
     - All Basic Tier features
     - Advanced query optimization
     - Priority email support
     - Dashboard analytics

3. **Enterprise Tier**: $100/month
   - Features:
     - All Pro Tier features
     - Custom integrations
     - Dedicated account manager
     - 24/7 support

---

## Customer Acquisition Cost (CAC) Range
- Estimated CAC: **$50 - $100** per user
  - Marketing spend, sales efforts, and onboarding costs contribute to this range.

---

## Lifetime Value (LTV) Estimate
- Average user lifespan: 24 months
- Average revenue per user (ARPU) for each tier:
  - Basic Tier: $10 * 24 = **$240**
  - Pro Tier: $30 * 24 = **$720**
  - Enterprise Tier: $100 * 24 = **$2400**

### LTV Calculation
- Assuming a mix of users:
  - 50% Basic, 30% Pro, 20% Enterprise
  - LTV = (0.5 * $240) + (0.3 * $720) + (0.2 * $2400) = **$120 + $216 + $480 = $816**

---

## Break-even Users Count
- Monthly fixed costs (development, marketing, support): **$5,000**
- Contribution margin per user (Price - Cost):
  - Basic Tier: $10 - $0.75 = **$9.25**
  - Pro Tier: $30 - $0.75 = **$29.25**
  - Enterprise Tier: $100 - $0.75 = **$99.25**

### Break-even Calculation
- Break-even users for each tier:
  - Basic Tier: $5,000 / $9.25 ≈ **541 users**
  - Pro Tier: $5,000 / $29.25 ≈ **171 users**
  - Enterprise Tier: $5,000 / $99.25 ≈ **50 users**

---

## Path to $10K MRR
- Target MRR: **$10,000**
- Pricing Tier Mix for $10K MRR:
  - **Basic Tier**: 800 users * $10 = $8,000
  - **Pro Tier**: 67 users * $30 = $2,010
  - **Enterprise Tier**: 10 users * $100 = $1,000

### Total Users Required
- Total users for this mix: 800 + 67 + 10 = **877 users** 

### Summary
To achieve $10K MRR, we need approximately **877 active users** distributed across the pricing tiers as outlined above.
